주요 기능 정리

🧠

AI 분석

성격분석, 채팅요약 등의 기능들에 AI분석을 활용함으로써 다른 봇들과는 다른 독창적인 체험이 가능하게 합니다

🎮

미니게임

타자게임, 넌센스 퀴즈 등 다채로운 게임 기능을 통해 채팅방에 소통률 버프를 줍니다

⚙️

자동화 기능

강퇴 알림, 아침 브리핑, 삭메알림 등 알잘딱 기능을 통해 채팅방 관리를 수월하게 합니다

📊

데이터 분석

채팅 순위, 레벨 등 오픈채팅방 내 데이터를 보여줍니다

🔍

기록 조회

날짜별 채팅 기록 조회, 특정 메시지 조회 등의 기능들을 통해 안전하고 깨끗한 채팅문화 조성을 돕습니다

💎

100% 무료
